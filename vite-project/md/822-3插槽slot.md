## slot

插槽就是子组件中的提供给父组件使用的一个占位符，用<slot></slot> 表示，父组件可以在这个占位符中填充任何模板代码，如 HTML、组件等，填充的内容会替换子组件的<slot></slot>标签。

### 匿名插槽

```ts
// 子组件放置一个插槽
<div>
  <slot></slot>
</div>
// 使用
<template v-slot>
   <div>  </div>
</template>
```

### 具名插槽

```ts
// 子组件放置一个插槽
<div>
  <slot name="header"></slot>
    <slot></slot>
  <slot name="footer"></slot>
</div>
// 使用
<template v-slot:header>
   <div>  </div>
</template>
// 插槽简写
<Dialog>
    <template #header>
       <div>1</div>
   </template>
   <template #default>
        <div>2</div>
       </template>
    <template #footer>
        <div>3</div>
    </template>
 </Dialog>
```
